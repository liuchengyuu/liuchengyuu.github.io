react state初始化问题

报错

大致：

不能再一个组件尚未mounted时调用 setState() 方法，这是一个空操作，但是可能会导致bug，所以， 直接给this.state 赋值，或者定义成state = {};

问题代码

render() 方法中如果使用了 this.state.docDetail，会导致编译报错

仔细检查代码

constructor 里使用了 setState，改为直接赋值后正常
